# mindora frontend

## env requirements

安装 node 环境（版本>= 22.0.0）：https://nodejs.org/zh-cn/download

## install dependencies

- 运行以下命令安装依赖

```bash
npm install -g pnpm
pnpm install
```

## run dev

- 运行 dev:pc 后在浏览器访问 http://localhost:5173

```bash
pnpm run dev:pc
```
